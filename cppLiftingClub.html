A website specifically built for the Cal Poly Pomona Power Lifting Club. Displays an interactive menu for members to resrve equipments to use. The admin dashboard provides administrators to manage users and see reservations created by the members of the club.

Further improvements are being worked on to enhance the following:

  • Security across the program. Such as hiding the firebase config files from client side.
  • Subscription management. Directing unsubscribed users to a different landing page.
  • Paid Firebase plan should allow dynamic control over reservations and other parts of the database with Firebase's Realtime Firestore.


    • Key Technology Used to Create the Application Inlude:

    AWS EC2 for deployment

    Github Actions for CI/CD workflow

    React and Express from NodeJS to create the Frontend and Backend

    Firebase connections to control authorization of user registration and logins
